直感的なスワイプ操作で写真・動画を簡単に整理できるiOSアプリ。

Siftは、直感的なスワイプ操作で写真や動画を「Keep」「削除」「お気に入り」に素早く整理できるアプリです。Swift 6とSwiftUIを採用し、モダンで美しいUIを提供します。

## 機能

- **スワイプで整理** — 右でKeep、左で削除、ダブルタップでお気に入り
- **動画・Live Photo** — 長押しでその場でプレビュー
- **まとめて削除** — 左スワイプでキューに追加し、一括削除
- **Undo** — 直前の操作をすぐに取り消し
- **フィルター** — 写真・動画・Live Photo・スクショで絞り込み
- **アルバム作成** — 整理した写真をiOSアルバムに追加
- **幅広いフォーマット対応** — 写真、動画、Live Photo、RAW、バースト
- **結果の永続化** — 整理の進捗は自動で保存
- **Tip Jar** — アプリ内課金で開発者を支援

## 要件

- iOS 18.0以上
- iPhoneのみ
- 縦向き（Portrait）のみ

## インストール

1. リポジトリをクローン

```bash
git clone https://github.com/iam74k4/Sift-iOS.git
cd Sift-iOS
```

2. Xcodeで `Sift.xcodeproj` を開く

3. Signing & Capabilitiesで開発チームを設定

4. 実機またはシミュレーターでビルド・実行
