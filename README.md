A photo and video organizer app with intuitive swipe gestures for iOS.

Sift allows you to quickly sort photos and videos into "Keep", "Delete", and "Favorites" using intuitive swipe gestures. Built with modern Swift 6 and SwiftUI for a beautiful, performant experience.

## Features

- **Swipe to sort** — Right to keep, left to delete, double tap for favorites
- **Video & Live Photo** — Long press to preview in place
- **Batch delete** — Queue items with left swipe, then delete all at once
- **Undo** — Revert any action instantly
- **Filter** — Narrow down by photos, videos, Live Photos, or screenshots
- **Album creation** — Add sorted photos to iOS albums
- **Full format support** — Photos, videos, Live Photos, RAW, and burst
- **Persistent results** — Sorting progress is saved automatically
- **Tip Jar** — Support the developer with optional in-app purchases

## Requirements

- iOS 18.0 or later
- iPhone only
- Portrait orientation only

## Installation

1. Clone the repository

```bash
git clone https://github.com/iam74k4/Sift-iOS.git
cd Sift-iOS
```

2. Open `Sift.xcodeproj` in Xcode

3. Configure your development team in Signing & Capabilities

4. Build and run on a device or simulator
